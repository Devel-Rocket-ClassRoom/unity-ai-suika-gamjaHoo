# 수박 게임 (Suika Game) — GDD

## 1. 게임 개요

| 항목 | 내용 |
|---|---|
| 장르 | 2D 물리 퍼즐 |
| 플랫폼 | PC (Unity 에디터 / WebGL) |
| 한 줄 요약 | 과일을 떨어뜨려 같은 종류끼리 합치고, 수박으로 키워라 |

**코어 루프**
1. 다음 과일 미리보기 확인
2. 마우스 X 위치로 떨어뜨릴 위치 조준
3. 클릭으로 과일 투하
4. 물리 연산으로 착지·구름
5. 같은 종류 접촉 → 다음 단계로 합체 + 점수 획득
6. 게임오버 라인 위에 과일이 일정 시간 머물면 종료

---

## 2. 핵심 메커닉

### 2-1. 투하
- 마우스 X 좌표로 떨어뜨릴 열(位置) 결정
- 클릭 시 현재 과일 스폰, 다음 과일이 큐에서 대기
- 투하 후 짧은 쿨다운(0.5s) 동안 재투하 불가

### 2-2. 물리
- Rigidbody2D + CircleCollider2D
- 중력으로 자연 낙하, 벽·바닥 튕김

### 2-3. 머지
- 같은 단계의 과일 두 개가 충돌 → 둘 다 제거 → 충돌 지점에 다음 단계 과일 생성
- 최고 단계(수박, 11단계)는 머지 없음
- 한 충돌 이벤트에서 중복 머지 방지 (플래그 처리)

### 2-4. 게임오버
- 컨테이너 상단에 트리거 라인 배치
- 과일이 라인 위에 **2초 이상** 머물면 게임오버
- 게임오버 시 결과 점수 표시 + 재시작 버튼

## 3. 과일 단계

| 단계 | 이름 | 상대 크기(반지름) | 머지 점수 |
|---|---|---|---|
| 1 | 체리 | 0.20 | 1 |
| 2 | 딸기 | 0.27 | 3 |
| 3 | 포도 | 0.34 | 6 |
| 4 | 귤 | 0.43 | 10 |
| 5 | 감 | 0.54 | 15 |
| 6 | 사과 | 0.67 | 21 |
| 7 | 배 | 0.83 | 28 |
| 8 | 복숭아 | 1.03 | 36 |
| 9 | 파인애플 | 1.27 | 45 |
| 10 | 메론 | 1.55 | 55 |
| 11 | 수박 | 1.90 | 66 |

- 투하 가능 과일: 1~5단계 (랜덤, 균등 확률)

---

## 4. 점수·승패 규칙

- **점수**: 머지 발생 시 해당 단계의 머지 점수 누적
- **게임오버**: 게임오버 라인 초과 2초 → 종료, 현재 점수 표시
- **승리 조건 없음** (최고 점수 경쟁)

## 5. 화면·UI 구성

```
┌─────────────────────────────┐
│  SCORE: 0000   [NEXT: 🍓]  │  ← 상단 HUD
├─────────────────────────────┤
│         ════════════        │  ← 게임오버 라인 (점선)
│                             │
│        [ 게임 영역 ]        │
│     벽           벽        │
│  ___________________________│  ← 바닥
└─────────────────────────────┘
```

| 요소 | 위치 | 설명 |
|---|---|---|
| 현재 점수 | 상단 좌측 | 실시간 갱신 |
| 다음 과일 미리보기 | 상단 우측 | 다음 투하 과일 스프라이트 |
| 드롭 인디케이터 | 마우스 X 정렬 | 떨어질 위치 점선 |
| 게임오버 패널 | 화면 중앙 | 점수 + 재시작 버튼 |

---

## 6. 톤 & 분위기

- **색감**: 밝고 채도 높은 파스텔 계열
- **아트 방향**: 플랫 카툰, 둥글고 귀여운 과일 일러스트
- **BGM**: 경쾌한 팝 (미구현 → 자유 구현 범위)
- **효과음**: 투하 · 머지 · 게임오버 (미구현 → 자유 구현 범위)
